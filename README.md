# Customer Churn Prediction 📊

A machine learning project focused on predicting customer churn using classification models. This project helps businesses identify customers who are likely to cancel a subscription or service, allowing for proactive retention strategies.

## 🗂️ Dataset
The dataset used contains customer demographic, account, and behavioral information. Key features include:
* **Tenure:** Number of months the customer has stayed with the company.
* **Monthly Charges:** The amount charged to the customer monthly.
* **Total Charges:** The total amount charged to the customer.
* **Churn:** Whether the customer churned or not (Target Variable).

## 🚀 Models Implemented
We explore and compare multiple classification algorithms to find the best performer:
1. **Logistic Regression** (Baseline model)
2. **Random Forest Classifier**
3. **XGBoost Classifier**
